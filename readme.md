# School Discord Bot
Good for verifying a user's email before giving access to the server, and giving roles for 7th-12th grade.
### Instructions
1. Copy the repo
`git clone svalencia014/schoolBot` 
or
`gh repo clone svalencia014/schoolBot`

2. Go into the directory
`cd schoolBot`

3. Install packages
`./setup.sh`

4. Run with node
`node .`
